# titanic-analysis
Exploratory data analysis of Titanic passenger data using Python and Pandas

I analysed a dataset containing information on 891 Titanic passengers, including age, class, port of embarkation and survival status. During cleaning, missing age values were replaced with the column mean, while the cabin column was dropped entirely due to insufficient data. The analysis revealed that women had a significantly higher survival rate than men (74% vs 19%), reflecting the "women and children first" policy. Passenger class also played a major role — first class passengers were more than twice as likely to survive as those in third class (63% vs 24%). Interestingly, age showed little difference between survivors and non-survivors, suggesting it was a weaker predictor of survival than gender or class.
Future analysis could explore whether fare price or embarkation port influenced survival.
